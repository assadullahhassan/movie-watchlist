# My Movie Watchlist

A responsive web application that allows users to search for movies using the OMDb API and maintain a personal watchlist. Users can search for titles, view movie details, and save their favorite picks to their browser's local storage.

## 🚀 Features

- **Dynamic Movie Search**: Integrated with the OMDb API to fetch real-time movie data.
- **Persistent Watchlist**: Uses `localStorage` to ensure your saved movies remain available even after refreshing the page.
- **Search Pagination**: Efficiently navigate through multiple pages of search results.
- **Add/Remove Functionality**: Seamlessly manage your list by adding new finds or removing movies you've already seen.
- **Responsive Design**: A mobile-friendly layout built with CSS Flexbox and Media Queries.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for better accessibility.
- **CSS3**: Custom styling with a focus on responsive layouts (Flexbox, Media Queries).
- **Vanilla JavaScript**: DOM manipulation, Fetch API for asynchronous requests, and application logic.
- **OMDb API**: External data source for movie information.
